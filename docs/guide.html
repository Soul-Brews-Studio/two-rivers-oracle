GitHub Pages Portfolio

4 ขั้นตอน — เว็บไซต์ portfolio ของตัวเอง

1
สร้างบัญชี GitHub
5 นาที
  • ไปที่ github.com แล้ว Sign Up
  • ตั้ง username ให้จำง่าย (จะเป็นชื่อเว็บเรา)
  • ใครมีแล้ว → Sign In ได้เลย
Tip: username ควรเป็นภาษาอังกฤษ ตัวเล็ก ไม่มีเว้นวรรค
2
สร้าง Repository
3 นาที
  • กดปุ่ม + มุมขวาบน → New repository
  • ชื่อ repo: username.github.io
  • เลือก Public
  • ไม่ต้องติ๊ก README → กด Create repository
Tip: ชื่อต้องตรงกับ username เป๊ะ เช่น somchai.github.io
3
อัปโหลด HTML
3 นาที
  • กด "uploading an existing file"
  • ลากไฟล์ portfolio HTML เข้าไป
  • เปลี่ยนชื่อเป็น index.html
  • กด Commit changes
Tip: ชื่อไฟล์ต้องเป็น index.html เท่านั้น ไม่งั้นเว็บจะไม่ขึ้น
4
เปิดเว็บไซต์
2 นาที
  • รอ 1-2 นาที
  • เปิด browser → พิมพ์ https://username.github.io
  • เห็น portfolio = สำเร็จ!
404? รอ 2-3 นาที แล้ว refresh อีกที / เช็คชื่อ repo + ชื่อไฟล์

สำเร็จแล้ว!

ทุกคนมีเว็บไซต์ portfolio ของตัวเองแล้วค่ะ

แชร์ลิงก์ให้ใครก็ได้ทั่วโลก
